# Контекст НИР

Тема НИР:
Разработка нейросетевой системы для прогнозирования кибератак и выявления аномального поведения пользователей на основе анализа сетевых логов и пользовательской активности.

Цель исследования:
Создать и обучить модель, объединяющую анализ внешних сетевых событий и внутренних поведенческих паттернов пользователей для раннего выявления потенциальных угроз безопасности.

Текущий этап:
Промежуточные результаты исследования.

Что нужно получить на текущем этапе:
- первый исследовательский Python-прототип;
- генерация или загрузка тестовых логов;
- предобработка сетевых и пользовательских событий;
- построение признаков;
- baseline-модель;
- LSTM-Autoencoder;
- первые метрики качества;
- графики;
- текстовое описание промежуточных результатов.

Выбранный подход:
- синтетический датасет сетевых и пользовательских логов;
- baseline: Isolation Forest и Random Forest;
- нейросетевая модель: LSTM-Autoencoder;
- метрики: precision, recall, F1-score, ROC-AUC, PR-AUC;
- графики: confusion matrix, ROC curve, reconstruction error distribution.

Ограничения текущего прототипа:
- данные синтетические;
- прототип не является промышленной IDS-системой;
- результаты являются предварительными;
- требуется дальнейшая проверка на открытых датасетах CICIDS2017, CSE-CIC-IDS2018 или CERT Insider Threat Dataset.

Связь с предыдущими этапами:
- в литературном обзоре обоснованы методы deep learning, log anomaly detection, UEBA, IDS и graph-based intrusion detection;
- в методологическом разделе выбран прикладной путь через последовательностные модели, LSTM-прогнозатор и LSTM-Autoencoder;
- следующий этап должен показать первые практические результаты реализации.
